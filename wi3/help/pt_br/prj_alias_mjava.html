Alias de Banco de Dados (MJava)


Para utilizar a conexão nativa MJava será necessário instalar o módulo MJava em um SGBD compatível com a tecnologia M. Para isso basta instalar com o comando %RIMF o pacote mjavaXXX.obj que acompanha a distribuição do WebIntegrator e executar a rotina ^MJAVA, para ativar o servidor MJava escolha a opção 1.

O WebIntegrator consegue comunicar-se com vários sistemas gerenciadores de bancos de dados M através do servidor MJava. A porta padrão que o servidor MJava fica escutando é a 3500. Para definir o campo Alias de uma conexão MJava basta indicar o nome ou IP da máquina e opcionalmente o número da porta e o usuário, veja exemplos a seguir:

//servidor
//servidor/user
//192.0.0.30:3500
//192.0.0.30:3500/user
