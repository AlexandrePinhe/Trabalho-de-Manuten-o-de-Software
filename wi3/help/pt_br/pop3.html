POP3


O tipo POP3 serve para que uma lista dos e-mails de um determinado usuário de uma conta num servidor POP3 seja gerada. As variáveis disponíveis ao desenvolvedor para exibir as informações são exibidas quando o usuário gera o arquivo do grid pela primeira vez. A saber:

Variável
Descrição
rowid
Retorna o identificador único de uma linha da tabela.
rowseq
Retorna o sequencial de uma linha da tabela.
subject
Retorna o valor do campo Subject (assunto) do e-mail.
size
Retorna o tamanho da mensagem em bytes.
date
Retorna a data que a mensagem foi recebida.
from
Retorna quem é(são) o(s) remetente(s) da mensagem.
