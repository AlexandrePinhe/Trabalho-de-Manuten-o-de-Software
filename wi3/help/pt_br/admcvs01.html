CVS


Este é o local onde o administrador do projeto poderá configurar o CVS para o projeto.

O campo Servidor indica o servidor de CVS a ser utilizado.

O campo Repositório indica o repositório a ser utilizado.

O campo Usuário indica o usuário do CVS.

O campo Senha indica a senha do usuário do CVS.
