Executando uma ação ao carregar uma página WSP


Se você quiser definir uma função JavaScript no atributo ONLOAD da tag BODY de uma página WSP gerada pelo WIzard insiram um elemento do tipo Genérico na página WSP e na propriedade Texto desse elemento coloque o seguinte trecho de código:

<script>
window.onload = function() {init(); <seu-código>};
</script>

A parte indicada por <seu-código> pode ser substituída por qualquer código JavaScript inclusive chamada a outras funções. A função init() é uma função usada pelas páginas geradas pelo WIzard para uma pré-formatação dos campos de um formulário então recomendamos que ela seja a primeira instrução a ser executada antes de qualquer código.
