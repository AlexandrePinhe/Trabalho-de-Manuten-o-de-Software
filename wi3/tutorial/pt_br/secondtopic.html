O que é o WebIntegrator?
Início  Anterior  Próximo

O WebIntegrator é um servidor de aplicações web que possui o seu próprio ambiente de desenvolvimento integrado (IDE) capaz de agilizar o desenvolvimento de aplicações web podendo acessar diversos tipos de fontes de dado oferecendo um modelo de programação onde a camada da regra de negócio (lógica) e a regra de apresentação são definidas separadamente facilitando a manutenção de um projeto e uma melhor distribuição de tarefas de uma equipe.

Focado na produtividade do desenvolvimento de aplicações web a IDE do WebIntegrator, o WI_Builder, necessita apenas de um browser para ser utilizada. Conectando-se com os principais bancos de dados do mercado via ODBC ou JDBC o WebIntegrator é indicado tanto para o desenvolvimento de novas aplicações quanto para a evolução de aplicações já existentes (sistemas legados) podendo se conectar nativamente com bases de dados MUMPS (MJava), BR/Search, etc.

Funcionalidades como desvio condicional, notificação por e-mail e anexação de documentos permitem que com o WebIntegrator sejam construídas aplicações típicas de workflow. Através de um mecanismo de conexão entre o cliente (browser) e o servidor de aplicações através de um applet, o WebIntegrator também pode executar transações com um banco de dados sem a submissão da página.
