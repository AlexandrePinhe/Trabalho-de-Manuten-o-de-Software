Para um correto aproveitamento desse tutorial sugerimos que o usuário possua um mínimo de conhecimento das tecnologias citadas a seguir:

·HTML (HiperText Markup Language): linguagem usada para a formatação visual (apresentação) de páginas serão exibidas por um browser. Um entendimento básico de comandos HTML para criar tabelas, manipular formulários e criar links seria necessário;  

·SQL (Structured Query Language): linguagem usada para manipulação de informações armazenadas em banco de dados relacionais. Alguma familiaridade com os comandos básicos do SQL como INSERT e SELECT ajudaria muito ao desenvolvedor na hora de criar aplicações no WebIntegrator que precisem interagir com um banco de dados relacional.  

Se você acha que não tem um conhecimento adequado das tecnologias mencionadas acima não se preocupe pois a medida que os comandos forem sendo apresentados uma breve explicação será fornecida juntamente com alguns links para que você possa aprofundar ou saber mais sobre o assunto.
