Este tutorial descreve o processo de desenvolvimento de uma aplicação web usando o servidor de aplicações WebIntegrator. Ele foi organizado de maneira a ofercer ao novo usuário do WebIntegrator um nível de aprendizado crescente e contínuo começando desde a criação e definição de um projeto até os passos para uma efetiva publicação em produção do projeto desenvolvido.

Ao final do tutorial o usuário deverá possuir os conhecimentos básicos e iniciais para que ele possa desenvolver um projeto através do WebIntegrator e saber como se configura os principais e mais comumente usados componentes utilizados em um projeto.
